# Markdown Structure Contract

This document defines the expected structure for Markdown files generated by the AI agents.

## File Structure

-   Each chapter will be a single `.md` file.
-   The filename will be the chapter title in kebab-case (e.g., `ros-2-core-concepts.md`).

## Content Structure

-   The file MUST begin with a level 1 heading (`#`) for the chapter title.
-   Sections MUST be level 2 headings (`##`).
-   Subsections MUST be level 3 headings (`###`).
-   Code blocks MUST be enclosed in triple backticks with the language specified (e.g., ` ```python `).
-   Citations MUST be rendered using the custom React component.
